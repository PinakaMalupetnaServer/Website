PMnS as Open Source

Server:
Our server is using PocketMine-MP, which is a PHP Based Minecraft Server Software, it is light and powerful Server Software than any other Minecraft Server Softwares around, PMMP is Open Source and uses LGPL-3.0 License.

PocketMine are not affiliated with Mojang. All brands and trademarks belong to their respective owners. PocketMine-MP is not a Mojang-approved software, nor is it associated with Mojang.

Plugins:
Our server are using open-sourced plugins from Poggit (PocketMine-MP), and we also make our own Plugins, Texture Packs for our Server, all of these are Open Sourced and available to our GitHub Organization Page.
